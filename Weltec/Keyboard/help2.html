Operating the keypad

    Shift Key

  • Numerical display - Involves the user clicking on the given key(shift), this automaticaly changes the appearence of the numeric keys. Eg: 1 now becomes !, 2 becomes @ and so on and so fourth
  • Key change - All keys on the fist row, starting from the ` right up to = key, change to the above character that we usually see on a convensional computer keyboard.
  • Sequence of change - Buttons can only be changed one at a time, this involves the user pressing shift first and then selecting any of the top row keys to change. The keys become reset once the focus of the pressed key becomes changed.

    • Space bar

      - Once this is clicked it creates a space, just like a normal computer would.

    • Caps lock

      - Works just like any other caps lock, once pressed all characters codes are displayed as uppercase.To reset it back to lowercase it is pressed again.
